# 钱包

#### EVM 兼容钱包 <a href="#evm-compatible-wallets" id="evm-compatible-wallets"></a>

为了与以太坊和其他 EVM 网络交互，Juchain 支持以下钱包：

* [MetaMask](https://metamask.io/) - 具有集成 dApp 支持的浏览器扩展。

*与 EVM 兼容的钱包允许您执行智能合约并与各种 EVM 网络上的分散应用程序进行交互。*
